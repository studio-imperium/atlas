# Atlas map generator

[![GoDoc](https://godoc.org/github.com/studio-imperium/atlas?status.svg)](https://godoc.org/github.com/studio-imperium/atlas)

Atlas is a 2d map generator which serves as a way to chunk of a world with voronoi diagrams, and also generate all sorts of terrain by assigning a "biome" to each cell.

### Installation

    go get github.com/studio-imperium/atlas
